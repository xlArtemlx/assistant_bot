# Assistant Bot

CLI personal assistant bot for managing contacts, birthdays, and notes.

## Installation

```bash
pip install -e .

```

## Run

assistant-bot

## Delete

```bash
pip uninstall assistant-bot -y

```

# Assistant Bot

CLI personal assistant written in Python.

The bot helps manage:

- contacts
- phone numbers
- emails
- addresses
- birthdays
- notes with tags

# Features

### Contacts

- Add new contacts
- Store phone numbers
- Store email
- Store address
- Store birthday
- Edit contact data
- Delete contacts
- Search contacts
- Show upcoming birthdays for a given number of days

### Notes

- Add notes
- Edit notes
- Delete notes
- Search notes
- Organize notes with tags
- List all tags
- Delete notes by tag

### Birthdays

- Add or update birthday
- Show contact birthday
- Show upcoming birthdays
